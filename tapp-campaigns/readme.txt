Enterprise-scale campaign management for internal teams. Create time-boxed product selection campaigns with WooCommerce integration.

== Description ==

TAPP Campaigns enables organizations to run time-boxed, invite-only product selection campaigns for internal teams.

**Key Features:**

* Create Team and Sales campaigns
* Time-boxed campaign management
* Invite-only participant system
* Product selection with color/size options
* WooCommerce integration
* Email notifications
* Real-time statistics
* CSV exports
* Role-based permissions (Manager, CEO, Staff)

**Perfect for:**

* Corporate Uniform Programs
* Sales Incentives
* Team Building
* Seasonal Campaigns
* Employee Benefits

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/tapp-campaigns/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure WooCommerce is installed and active
4. Navigate to Campaigns menu to get started

== Frequently Asked Questions ==

= Does this require WooCommerce? =

Yes, TAPP Campaigns requires WooCommerce to be installed and active.

= Can I customize email templates? =

Yes, email templates can be customized in the Settings page.

= How many participants can I add? =

The plugin is designed to handle 1,000+ participants per campaign.
